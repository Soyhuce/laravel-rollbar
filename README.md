# Rollbar wrapper for Laravel

![Release](https://img.shields.io/badge/Release-x.x.x-blue.svg)

## Installation

You can install the package via composer:

```bash
composer require soyhuce/laravel-rollbar
```

## Testing

```bash
composer test
```
